Usage

1

Include Dependencies

<script src="https://code.jquery.com/jquery-3.7.1.min.js"></script>
<script src="/path/to/velisoft-img-lib.js"></script>
<link rel="stylesheet" href="/path/to/velisoft-img-lib.css"></link>
2

Setup HTML Structure

<div data-velisoft-img-lib="gallery" data-velisoft-img-libcontrols="true">
    <img src="/img1.jpg" alt="Image 1" class="gallery_image">
    <img src="/img2.jpg" alt="Image 2" class="gallery_image">
    <img src="/img3.jpg" alt="Image 3" class="gallery_image">
</div>
3

Initialize

The library automatically initializes when the DOM is ready. Just add the required attributes and classes!

*

Important

Using the scroll behavior css will cause the library to scroll back to the top when closing the container.
So revert from using this! 

html{scroll-behavior: smooth;}

Documentation

Attributes

data-velisoft-img-lib

Set to "gallery" or "single" to define the viewing mode

data-velisoft-img-libcontrols

Set to "true" or "false" to show/hide UI controls

class="gallery_image"

Required class for images to be included in the library

Controls

Keyboard:

Arrow keys for navigation, ESC to close, Space for autoplay

Mouse:

Click and drag to pan, scroll to zoom, swipe to navigate

Touch:

Pinch to zoom, swipe to navigate, drag to pan
